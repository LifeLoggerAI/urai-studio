# URAI Studio Public Copy Guardrails

Use these guardrails for public pages, launch copy, and demo surfaces.

## Say clearly

- Public shell is online.
- System contracts and status routes are available.
- Admin and QA controls are gated.
- Generation, billing, private downloads, dashboards, and integrations depend on configured backends and verified access.
- V1 includes ground, orb, chat, magical home shell, memory stars, emotional weather, relationship fields, recovery fields, and field reconstruction.

## Avoid saying unless verified

- All systems operational.
- Backend connected.
- Generation is live.
- Admin diagnostics are public.
- Billing is enabled.
- Private downloads are ready.
- User memory is live.

## Public route posture

- `/` and `/demo` may show seeded product experiences.
- `/status` should link to public health and contract endpoints without exposing raw operator controls.
- `/admin` and `/studio/admin` must remain gated for public demos.
- `/contact` and `/waitlist` must describe production activation as conditional on verified configuration and access.
