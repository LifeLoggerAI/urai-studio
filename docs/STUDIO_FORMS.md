# Studio Forms

Every public Studio form needs:

- privacy note
- sourceDomain
- sourcePath
- utmSource
- utmMedium
- utmCampaign
- utmContent
- utmTerm
- referrer
- loading state
- success state
- error state

Project briefs should route to projectRequests or the approved backend equivalent.

Do not collect sensitive personal data unless it is required for the project inquiry.
